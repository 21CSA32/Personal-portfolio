# Litheeshkumar G - DevOps Engineer Portfolio

## Overview
This is a personal portfolio website for Litheeshkumar G, a Cloud & DevOps Engineer specializing in AWS, GCP, Docker, Kubernetes, CI/CD, and Cloud Infrastructure Management.

## Project Type
Static HTML/CSS/JavaScript website

## Structure
- `index.html` - Home page with hero section and quick stats
- `about.html` - About page
- `skills.html` - Skills showcase
- `projects.html` - Projects portfolio
- `contact.html` - Contact form
- `css/style.css` - Stylesheet with responsive design and dark mode
- `js/script.js` - JavaScript for interactivity (smooth scroll, dark mode toggle, form validation)
- `images/` - Directory for images (profile photo, etc.)

## Technology Stack
- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- External libraries: Font Awesome, Google Fonts (Poppins)
- Server: Python 3.11 HTTP server for development

## Features
- Responsive design
- Dark mode toggle
- Smooth scrolling
- Form validation
- Active navigation highlighting
- Scroll-to-top button
- Scroll animations

## Setup & Configuration
- Development server runs on port 5000
- Server configured with cache-control headers for Replit environment
- Static file serving via Python's http.server module
